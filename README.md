# Mousikal-client

The client for a music discussion web application using Vue.js.

### Running the project
```
<!-- Install project files -->
npm install 

<!-- Start a development server -->
npm run serve

<!-- Build the application to deploy -->
npm run build

<!-- Run linting and fixes -->
npm run lint
```
